# Cleanup pass notes

Session-derived pattern for light inbox polishing.

## What changed well
- Added `Category` and `Tags` to drafts that only had a raw notes block.
- Split long notes into short readable lines.
- Preserved the original tone / uncertainty instead of over-editing.
- Added `Related places` where the relationship is obvious and useful for future graph mapping.

## Good relation examples
- `Bowlarama Bathurst Bowling & Arcade` ↔ `Big D Drive-In`
- `CCNB Bathurst Campus — 725 College St` ↔ `CCNB Bathurst Campus — 75 Youghall Dr`
- `Apollo Cinema` ↔ nearby dine-in options
- `Bootlegger` ↔ `La Louga Night Club` / waterfront corridor

## Cleanup rule of thumb
- Do not turn draft cleanup into final-page normalization.
- Add structure only when it makes search or clustering easier.
- Preserve uncertainty in notes instead of inventing precise facts.
- If the draft already has useful raw phrasing, keep it.
