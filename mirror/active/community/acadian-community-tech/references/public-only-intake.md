# Public-only directory intake

Session note: when drafting Acadie.sol directory entries, only collect public-facing business data:
- address
- public phone number
- public business hours
- email only if publicly posted

Pattern discovered:
1. start with a small batch of names
2. look up public sources
3. extract a minimal, fixed field set
4. stage the result in `inbox/` as a draft
5. later split into `entries/<slug>/` when the page shape is clear

Useful distinctions:
- keep location pages separate from future brand/collective pages when a business has multiple branches
- preserve the source URL in the draft so later normalization is traceable
- if a public email appears later, add it to the draft/entry without expanding the field set beyond public data

Example batch from Acadie-Bathurst:
- Chicken King & Chinese Palace
- Big D Drive-in
- Big Deal Produce Market Ltd
- Pizza Shack (Bathurst location only)
