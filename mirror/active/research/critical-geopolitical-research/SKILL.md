# Critical Geopolitical Research & Pattern Analysis

## Trigger
User asks for deep dive into geopolitical conflicts, questioning official narratives, or seeking to understand patterns across multiple interventions (e.g., Iran, Israel/Palestine, Venezuela, Iraq, Afghanistan). User expresses skepticism about media coverage and wants verified facts, historical context, and analysis of propaganda patterns.

## Approach

### 1. Multi-Angle Source Gathering
- Search current news from diverse outlets (Al Jazeera, Reuters, NYT, independent journalists)
- Include perspectives from affected countries, not just Western media
- Look for academic/think tank analysis (CFR, Brookings, Pew)
- Find voices from the actual populations being discussed

### 2. Historical Pattern Recognition
- Trace roots back to key historical moments (e.g., 1948 for Middle East)
- Compare to previous US interventions (Vietnam, Iraq, Afghanistan)
- Identify recurring justifications and outcomes
- Map propaganda techniques across different conflicts

### 3. Humanization Focus
- Counter dehumanizing narratives with cultural context
- Find ordinary people's voices, not just government statements
- Document what's missing from Western coverage
- Highlight family structures, education, arts, daily life

### 4. Economic Interest Analysis
- Follow the money: oil, contracts, strategic resources
- Compare treatment of allies vs. targets (e.g., Saudi Arabia vs. Iran)
- Identify who benefits economically from conflict
- Track sanctions impact on civilians vs. governments

### 5. Synthesis & Documentation
- Create comprehensive document with sources
- Use tables for pattern comparison
- Include direct quotes from primary sources
- Organize by theme, not chronology
- End with actionable questions for critical thinking

## Output Format
Save to user's notes system with:
- Clear section headers
- Source citations for all claims
- Timeline where relevant
- Comparison tables
- Key questions for ongoing analysis
- Action steps for user

## Pitfalls
- Don't present as "truth" but as "verified facts from multiple sources"
- Avoid conspiracy language; stick to documented patterns
- Include mainstream sources alongside independent ones
- Acknowledge complexity; avoid oversimplification
- Don't equate governments with populations
- Remember: user wants facts to form their own conclusions

## Verification Steps
1. Cross-reference claims across 3+ sources
2. Check dates and context of quotes
3. Verify think tank political leanings
4. Include sources user can independently verify
5. Distinguish between documented facts and analysis
