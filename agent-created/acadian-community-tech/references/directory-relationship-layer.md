# Directory relationship layer notes

Use this when turning inbox drafts into clean entries for the Acadie.sol directory.

## What to preserve from raw notes
Capture and later normalize adjacency and context clues such as:
- next to / beside / across from / in front of
- close range to / near / on the same street as
- same corridor / same strip / same waterfront / same downtown block
- shared events or recurring community tie-ins
- sibling businesses or neighboring anchors

## Suggested clean-entry fields
- `related_places`
- `nearby_places`
- `shared_corridor`
- `shared_events`
- `location_context`

## Useful patterns from the Bathurst cluster
- Downtown hospitality strip around Main St can be modeled as a corridor.
- Waterfront breweries can be linked as a separate cluster.
- Event anchors (for example seasonal block parties or hospitality days) deserve backlinks so the directory reflects the social graph, not just addresses.

## Rule of thumb
Preserve the raw wording in the inbox draft, then add the graph layer only once the clean entry is being formed.
