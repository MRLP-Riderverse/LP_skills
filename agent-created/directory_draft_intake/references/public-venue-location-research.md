# Public Venue / Location Research Notes

Use this when a place has multiple branches or lives inside a shared venue cluster like a mall, marina, plaza, or waterfront district.

## What worked

- Search the exact place name plus a landmark or branch label:
  - `"Greenhouse Lounge" "Place Bathurst Mall"`
  - `"Greenhouse Lounge" "Bathurst Marina"`
  - `"Le Greenhouse Lounge" Bathurst`
- Check multiple source types, not just one:
  - official mall / venue pages
  - tourism listings
  - Facebook or Instagram pages
  - contact pages
  - event listings
- Use result snippets when they clearly expose branch labels, addresses, or seasonal status.
- Keep one draft per physical location.
- Preserve the official branch label in Notes if it differs from the user-facing title.

## Example branch split

- `Greenhouse Lounge - Place Bathurst Mall`
  - official mall listing at `1300 St Peter Ave`
- `Greenhouse Lounge - Marina`
  - Bathurst Marina listing at `1555 Queen Elizabeth Drive`
  - tourism listing notes it is closed for winter

## Rule of thumb

If two public listings share a brand name but point to different physical places, do not merge them into one entry. Stage separate drafts and disambiguate the titles by branch or address.
