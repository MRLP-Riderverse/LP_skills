# Public source verification notes

Use this when a draft needs a small amount of source-backed enrichment before it is exported.

## Preferred source order
1. Official municipal / civic / parks pages
2. Official brand or location pages
3. Search snippets or map listings as fallback

## What to extract from official city/service pages
- operating hours
- parking
- access rules
- designated zones
- rentals / rentals operator
- lifeguards / seasonal operations
- pet / camping / fishing restrictions
- cleaning / maintenance schedules
- wheelchair / accessibility notes

## Capture rule
- Put verified facts in `## Notes` or `## Public source`
- Keep uncertain-but-useful claims out of public copy
- Mark unconfirmed safety/rule claims in `## Admin notes` as unverified

## Example pattern
- Good: “City page says free parking, lifeguards 10:00am–5:30pm, and no overnight camping.”
- Avoid: “I believe zero fire tolerance” unless a source explicitly says so.
